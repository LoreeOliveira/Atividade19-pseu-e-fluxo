# Atividades de Lógica de Programação - SENAI 💻

Este repositório contém os exercícios práticos desenvolvidos para o curso de **Desenvolvimento de Sistemas** no SENAI. O objetivo principal foi consolidar conceitos de lógica de programação através da criação de algoritmos e representações visuais.

## 🚀 Sobre o Projeto

As atividades foram desenvolvidas em dupla e consistem na resolução de problemas lógicos utilizando duas abordagens complementares:
1.  **Pseudocódigo:** Desenvolvido no software **VisualG** (arquivos `.alg`).
2.  **Fluxograma:** Modelado no software **Flowgorithm** (arquivos `.fprg`).

## 🛠️ Tecnologias Utilizadas

* [VisualG](http://visualg3.com.br/)
* [Flowgorithm](http://www.flowgorithm.org/)
* Git & GitHub para versionamento

## 📂 Exercícios Implementados

Com base nos requisitos da atividade, os seguintes problemas foram solucionados:

* **Bônus:** Lógica para cálculo de premiações ou benefícios.
* **Candidato:** Verificação de dados ou elegibilidade de candidatos.
* **Estoque:** Sistema simples para controle ou verificação de produtos.
* **Ingresso:** Lógica voltada para venda ou validação de entradas.
* **Nota e Frequência:** Sistema para calcular a média e verificar a presença do aluno (Aprovado/Reprovado).
* **Salário do Cliente:** Cálculo de remuneração líquida, descontos ou proventos.
